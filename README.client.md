# TutorBot_Server

TutorBot_Server requires the following files to be present in the
folder:

- static
- .env.example
- classes
- TutorBot_Server executable

To run this application using the `TutorBot_Server` executable,
follow these prerequisites.

## Prerequisites

### Set environmental variables

To set the required secrets, duplicate `.env.example`, rename it to
`.env`, and fill its contents with the corresponding secrets.

### Classes

The `classes` directory must be present and populated with the lesson
content the server will load at runtime.
